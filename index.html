New Castle County COVID-19 Dashboard
(R Shiny)

In June of 2020, Delaware's New Castle County (NCCo) Executive Matt Meyer asked CompassRed for a better visualization solution for the wastewater COVID-19 testing program. The initial data was shared to the public in a very barebones Excel document featuring just a data table and a (confusing) bar chart.Alongside a project manager, I led the NCCo team through a 3 week sprint to prototype and build a Shiny app that included an interactive leaflet visualization of the catchment areas being tested. Gathering app requirements and data details in such a short window was exciting but taught me that no matter the time crunch, the stakeholder management process for data products follows the same path!I consider this project a success because thousands of New Castle County residents were able to understand the visual representation of an innovative new program being used to track COVID-19 days before cases were reported.

Delaware Data Innovation Lab: Project Ditto
(R Shiny)

This project was built in partnership with the data science team at the Delaware Data Innovation Lab (now part of Tech Impact). The pitch was to build a tool that showed similarity in COVID-19 case trends by county.As the lead Shiny engineer on the project, I was responsible for wireframing and prototyping a solution, development and some data engineering, as well as conducting feedback sessions. One major challenge was the long re-render time of the choropleth map when the selected county was updated. To overcome this, I implemented some custom JavaScript to update the map directly in the browser (vs. using the Shiny server to re-render the entire map).Although this project was never included in the lab‚Äôs portfolio, I consider this a project a success because it was a unique experience in trying different solutions to build a useful tool.

What's in My YouTube History: Music
(React.js)

In order to cement my React.js learning, I knew I wanted to use my YouTube history data from Google Takeout after some Reddit inspiration. After some EDA in R using the tidyverse, I quickly I realized there was potential for a data app.In building this, I dove deep into the visx visualization module from Airbnb, the MUI module for material UI, and the tidyjs data transformation module (among others). I consider this project a success because the final app would not have been as interactive if I were to use other tools in my repertoire like Shiny or Power BI. Go JavaScript!

The Office Multi-Chart (Svelte/d3)

Svelte is a newer front-end JavaScript framework for designing great UI experiences. I'm learning that using d3.js is a breeze in Svelte! I wanted to find a dataset to make use of these tools and since I've been rewatching a lot of The Office recently, I ended up with this data app!I was able to leverage some previously created design patterns to develop this "interaction-rich" data app. I made use of the mark.js library for highlighting text and some of the native Svelte animations/transitions for all of the motion you see in the app. I consider this project a success because I was able to sync the visualizations in unique ways thanks to these new tools I'm learning!

Self-Care Analytics Infographic
(Looker Studio)

As part of the CompassRed blog, the team was asked to create data-related content that would not only attract visitors to the site and services, but build content that we were genuinely curious to explore and share! As someone who believes strongly in the benefits of mental health care, I explored what Google search trends were looking like shortly after the start of COVID-19 lockdown (April 2020).I consider this project a success because I was proud of the design and insights as well as some news coverage!

Trillionaire Mindset Podcast Data Mining

One of my favorite comedy podcasts (with some finance sprinkled in) is Trillionaire Mindset. For their 50th episode, I downloaded all of the podcast transcripts and use the tf-idf algorithm to surface the most "unique" words per episode. The result is a nice summary of the topics unique to each episode.Using ggplot2 in R, I was able to create a layout I was very pleased with! I consider this side project a success because the hosts Ben & Emil brought up the infographic on the show! I was honored.
